AI-SESSION-BRIDGE — Bridge AI coding sessions between Codex CLI and Claude Code
=================================================================================

PURPOSE: Bidirectional JSONL session converter between OpenAI Codex CLI and
         Anthropic Claude Code CLI. Enables continuing a coding session started
         in one tool using the other.

LOCATION: This is the ai-session-bridge project root directory.
TOOL: src/cli.ts (run with: npx tsx src/cli.ts)

COMMANDS:
  list [codex|claude]           — List recent sessions with message previews
  info <id|file>                — Show session metadata and record type counts
  preview <id|file>             — Show first messages from a session
  codex2claude <id|file>        — Bridge Codex session -> Claude Code format
  claude2codex <id|file>        — Bridge Claude Code session -> Codex format
  auto <id|file>                — Auto-detect source format and bridge

FLAGS:
  --json         Machine-readable JSON output (all commands)
  --dry-run      Show conversion stats without writing
  --output/-o    Custom output file path
  --help/-h      Show help

SESSION LOCATIONS:
  Codex CLI originals:     ~/.codex/sessions/YYYY/MM/DD/rollout-*.jsonl
  Claude Code originals:   ~/.claude/projects/-{PROJECT_PATH}/*.jsonl
  Bridged (Codex->Claude): ~/.claude/projects/-converted-from-codex/*.jsonl
  Bridged (Claude->Codex): ~/.codex/sessions/YYYY/MM/DD/converted-*.jsonl

SESSION ID RESOLUTION:
  Accepts full UUID, partial UUID (8+ chars), or absolute file path.
  Original sessions are prioritized over converted/bridged ones.

TOOL NAME MAPPING:
  exec_command <-> Bash
  read_file    <-> Read
  write_file   <-> Write
  patch_file   <-> Edit
  list_directory <-> Glob
  search_files <-> Grep
  request_user_input <-> AskUserQuestion

LOSSY FIELDS:
  - session_meta.base_instructions (Codex system prompt)
  - developer role messages (prefixed with [SYSTEM/DEVELOPER])
  - turn_context.collaboration_mode (Codex-specific)
  - file-history-snapshot (Claude Code-specific)
  - compacted summaries (expanded from replacement_history)

REQUIREMENTS: Node.js 18+, tsx
